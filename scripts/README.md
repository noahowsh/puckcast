# Scripts Overview

- `predict_fast.py` / `predict_full.py`: generate web feed (`web/src/data/todaysPredictions.json`).
- `scripts/twitter_ab_testing.py` + `scripts/post_to_twitter.py`: X automation (hashtags/mentions auto-added).
- `generate_insights.py`: build site metrics/model insights.
- `scripts/validate_predictions.py`: sanity checks predictions payload.

Deprecated/legacy scripts live under `archive/` or `analysis/`.
